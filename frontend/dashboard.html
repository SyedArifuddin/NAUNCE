    System Architecture

    Database Schema

    • Users: profiles and primary cultural languages.
    • Analyses: original text, JSONB tone/literal data, adaptability scores.
    • User Idioms: frequency tracking for marker clouds.
    • Chat History: contextual logs for Naunce Bot.

    API Endpoints

    • /api/auth/ registration, login, session management.
    • /api/analyze/ text and file processing logic.
    • /api/dashboard/ trends and marker analytics.
    • /api/chat/ contextual AI assistant messaging.
